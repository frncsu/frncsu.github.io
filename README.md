## Projects

### 1. Hiking Capstone — Trail Discovery & Outdoor Recreation System

*Overview:*
A Web/Mobile Application built as a capstone project for tourism, outdoor recreation, and community guiding. The system empowers hikers and tourists to discover, plan, and safely navigate trails while enabling administrators to manage bookings, permits, and guides in one centralized platform.

*Tech Stack:*
- *Type:* Web / Mobile Application (Capstone Project)
- *Maps Integration:* Google Maps / Mapbox
- *Goal:* Tourism, Safety & Community

*Key Features:*
- Browse, search, and filter hiking trails by difficulty, location, and distance
- Interactive trail maps with route tracking
- Permit management, tour guide bookings, and slot reservations
- Real-time weather updates and emergency SOS button
- Packing checklists and user reviews/ratings for trails

### 2. Employee Management System — HR & Back-Office Internal Tool

*Overview:*
An Enterprise Human Resource Management System (HRMS) designed to centralize and streamline employee data, attendance tracking, and payroll operations. Built as a classic CRUD application with a focus on clean database management and solid business logic.

*Tech Stack:*
- *Type:* Enterprise / HRMS Web Application
- *Focus:* Database Management, Business Logic
- *Goal:* Efficiency & Organization

*Key Features:*
- Centralized digital employee profiles (personal info, job titles, departments)
- Daily attendance and time tracking — clock-ins, clock-outs, working hours
- Leave and vacation request management
- Payslip generation based on hours worked
- Manager tools for performance evaluations and departmental tracking
